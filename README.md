# Бизнес Соло — Микро CRM

Локальное приложение для управления клиентами, задачами и финансами самозанятых и небольших предпринимателей. Работает полностью на устройстве, без передачи данных на сервер.

## Основные возможности
- 👥 **Клиенты** — учёт физических и юридических лиц, контактов и заметок.
- 📋 **Задачи (обязательства)** — предоплаты, постоплаты, затраты, сроки, просроченные задачи.
- 📊 **Воронка продаж** — настраиваемые стадии, перемещение клиентов, мусорка со статистикой отвалов.
- 💰 **Бухгалтерия** — доходы, затраты, автоматический расчёт налога НПД (4% / 6%), фильтрация по периоду.
- 🔒 **PIN-защита** — 4-значный код, восстановление через резервный код.
- 💾 **Резервное копирование** — экспорт/импорт всех данных в JSON, выгрузка CSV.
- 🌓 **Тёмная/светлая тема**, авто‑переключение по времени.

## Для кого
- Самозанятые (НПД)
- Индивидуальные предприниматели
- Фрилансеры, мастера, частные специалисты
- Все, кто хочет вести учёта клиентов и обязательств без облаков и подписок.

## Установка
- **APK (Android)**: (СКОРО!) скачайте последнюю версию из [раздела Releases](https://github.com/Ilyasmusic/ilyascrm/releases) и установите на устройство.
- **PWA (веб-версия)**: откройте [`index.html`](https://ilyasmusic.github.io/ilyascrm/) в любом современном браузере. Для офлайн‑доступа можно добавить сайт на домашний экран.

> Приложение сохраняет все данные в локальном хранилище браузера (localStorage). Для сохранности данных регулярно создавайте резервные копии.
