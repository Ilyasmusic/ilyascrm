👥 Клиенты

Чтобы добавить клиента, нажмите кнопку + в правом верхнем углу. Заполните ФИО (обязательно), тип (физлицо/юрлицо), контакты и описание. Кнопка ✏️ редактирует клиента, 🗑 удаляет. Поиск работает по ФИО.

📋 Задачи (обязательства)

Нажмите + в задачах, чтобы создать новое обязательство. Выберите клиента, описание, укажите предоплату/постоплату, затраты и дату. Кнопки Прд и Пст отмечают получение денег (зелёным). Просроченные задачи подсвечиваются красной полосой слева.

📊 Воронка продаж

Добавьте клиента в воронку через кнопку + Клиент. Перемещайте его между стадиями стрелками. Кнопка 🗑 удаляет в мусорку. В мусорке можно восстановить клиента или очистить всё. Управление стадиями — кнопка внизу воронки.

💰 Бухгалтерия

Здесь считаются ваши доходы, затраты и налог НПД (4% для физлиц, 6% для юрлиц/ИП). Выберите период (месяц, год, произвольный). Налог вычитается из чистой прибыли автоматически. Кнопка «Экспорт CSV» в настройках сохранит детализацию.

💾 Резервное копирование

Скачайте копию всех данных через кнопку Скачать. Загрузите файл JSON обратно, чтобы восстановить данные. Перед включением PIN-кода обязательно сделайте бэкап!

🔒 PIN-код

Включите защиту в настройках. Задайте 4-значный код. После трёх неверных попыток появится кнопка полной очистки. Код восстановления (8 цифр) показывается при включении — сохраните его, чтобы сбросить PIN.
